# sports gambling data

With the rise of sports gambling, mostly online, I was intriged about the data behind it. Specifically, just how much the betting giants profit off their customers, how much money the states make off of the taxes on betting, and if the benifits gained outweigh the costs. There are many variables in my dataset that I have collected to try and make some prediction models on a state by state basis. The lack of rows in the set makes it hard to create accurate models but I tried regardless. I want to continue this project into seeing the stats about underage online gamblings, which is personally witnessed quite frequently. Additionally, I believe the advertisements on online gambling have gotten out of line, too many and too targeted. Without a model, I predict that much controversy will be coming out about these topics in the near future. And when that happens, I hope to be able to use my data and models that I've created in this project to help solve problems in this later controversy.
